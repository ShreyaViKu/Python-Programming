# 🐍 Python Programming

A comprehensive collection of **Python programming** examples, practice programs, and coding exercises covering concepts from **basic to advanced**. This repository is created to help beginners like me to build a strong foundation in Python while improving logical thinking and problem-solving skills.

## 📚 Topics Covered

### 🔹 Python Fundamentals

* Introduction to Python
* Variables & Data Types
* Input & Output
* Operators
* Type Casting
* Comments

### 🔹 Control Flow

* Conditional Statements (`if`, `if-else`, `elif`, Nested `if`)
* Loops (`for`, `while`)
* Pattern Printing
* Loop Control Statements (`break`, `continue`, `pass`)

### 🔹 Functions

* User-Defined Functions
* Recursion
* Lambda Functions
* Variable Scope
* Built-in Functions

### 🔹 Number & String Programs

* Prime, Perfect, Armstrong, Palindrome, Strong Numbers
* Factorial & Fibonacci
* Factors & Divisors
* String Manipulation
* Pattern & Logic-Based Problems

### 🔹 Python Collections

* Lists
* Tuples
* Sets
* Dictionaries
* List Comprehensions

### 🔹 Advanced Python

* File Handling
* Exception Handling
* Object-Oriented Programming (OOP)
* Modules & Packages
* Iterators & Generators
* Regular Expressions (Regex)

### 🔹 DSA & Logic Building

* Searching Algorithms
* Sorting Algorithms
* Arrays & Strings
* Stacks & Queues
* Linked Lists
* Recursion
* Problem Solving & Coding Practice

## 🎯 Purpose

* Learn Python from basic to advanced
* Build strong programming fundamentals
* Improve logical thinking and coding skills
* Practice Data Structures & Algorithms using Python
* Prepare for coding interviews, college practicals, and technical assessments

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/ShreyaViKu/Python-Programming.git
```

Run any Python program:

```bash
python filename.py
```
